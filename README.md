## 这是什么
就是一个很简单的博客，使用 Hugo 生成静态网页，直接把模板 [Bilberry Hugo Theme](https://github.com/Lednerb/bilberry-hugo-theme) 挪过来用，顺便改了一些东西。   
内容的话大多数是自己的折腾记录。更新频率预估会很低 ~~其实是懒~~ ，以及不定期更新自己喜欢的音乐。
